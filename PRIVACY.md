# Política de Privacidade — Reclaim.ai Smart New Tab

Última atualização: 2024-11-20

## Dados coletados
- A extensão **não coleta, armazena nem compartilha dados pessoais**.
- Não usamos cookies, rastreadores ou analytics.

## Permissões
- `tabs`: servem apenas para verificar se já existe uma aba do Reclaim Planner aberta e, se não houver, atualizar a nova aba para o Planner.
- `*://app.reclaim.ai/*`: usado somente para navegar até o site do Reclaim. Não injetamos scripts na página nem lemos dados dela.

## Código remoto
- Não executamos código remoto. Todo código da extensão está empacotado. Apenas abrimos uma navegação para `https://app.reclaim.ai/planner`.

## Compartilhamento com terceiros
- Não compartilhamos dados com terceiros.

## Contato
- Para dúvidas ou solicitações, entre em contato pelo e-mail do desenvolvedor cadastrado na loja.
